## 🎯 구현 방식 (OpenClaw 제약사항)

**해결 방법**:
1. **Primary 모델**: Claude Sonnet 4.5 유지 (대화/코딩 품질 보장)
2. **내부 판단**: 위 전략에 따라 적절한 모델 선택
3. **sessions_spawn**: 서브에이전트 생성 시 모델 명시
4. **메모리 참조**: 이 파일을 항상 참고하여 최적 모델 사용

**예시**:
- 사용자가 "정부 지원 프로그램 조사해줘" → Gemini 사용 (무료)
- 사용자가 "이 코드 리뷰해줘" → Claude 사용 (품질)
- 사용자가 "이 문서 요약해줘" → Gemini 사용 (무료)
