## 🎯 가능한 음성 대화 방식

### 옵션 1: Telegram 음성 메시지 (추천 ⭐⭐⭐)

**가장 간단하고 즉시 사용 가능!**

```
위니엘님 (Telegram)
  ↓ 음성 메시지 녹음 (버튼 길게 누르기)
Telegram
  ↓ 자동 STT (Speech-to-Text) 변환
엘케인 (OpenClaw)
  ↓ 텍스트 처리 + 응답 생성
  ↓ TTS (Text-to-Speech) 변환
Telegram
  ↓ 음성 메시지로 전송 (동그란 버블)
위니엘님
  ↓ 음성 응답 듣기 🎧
```

**장점**:
- ✅ **지금 바로 사용 가능** (설정 5분)
- ✅ Telegram이 자동으로 음성→텍스트 변환
- ✅ 엘케인이 음성으로 응답
- ✅ 무료 (Edge TTS 사용 시)
- ✅ 비동기 대화 (언제든 재생 가능)

**단점**:
- ❌ 실시간 대화는 아님 (녹음 → 전송 → 응답)
- ❌ Telegram이 STT 해야 함 (인터넷 필요)

---

### 옵션 2: 실시간 음성 대화 - OpenClaw Voice (고급)

**ChatGPT Voice처럼 실시간 대화!**

```
위니엘님 (브라우저)
  ↓ 마이크로 말하기 (실시간)
WebRTC
  ↓ 음성 스트리밍
OpenClaw Voice (localhost)
  ↓ STT (Whisper) 실시간 변환
엘케인 (OpenClaw)
  ↓ 응답 생성
  ↓ TTS (ElevenLabs/OpenAI/Edge)
브라우저
  ↓ 즉시 음성 재생 🔊
위니엘님
  ↓ 실시간 대화 계속
```

**장점**:
- ✅ **실시간 대화** (말하면 바로 응답)
- ✅ ChatGPT Voice 같은 경험
- ✅ 자연스러운 대화 흐름
- ✅ 끊김 없는 대화 (think → speak loop)

**단점**:
- ❌ 추가 설정 필요 (15분~30분)
- ❌ Whisper API 비용 ($0.006/분)
- ❌ 브라우저 필요 (모바일 제한적)

- ✅ 모바일/유선 전화 모두 가능

- ❌ Twilio 비용 ($0.0085/분 + 전화번호)
- ❌ 설정 복잡 (30분~1시간)

## ⚙️ 옵션 1 설정 가이드 (Telegram 음성 메시지)

### 1단계: TTS 활성화 (5분)

#### 방법 A: 무료 (Edge TTS) - 추천

```bash
# 1. config 파일 편집
nano /home/winielab/.openclaw/openclaw.json

# 2. messages 섹션에 tts 추가
{
  "messages": {
    "tts": {
      "auto": "always",           // 항상 음성 응답
      "provider": "edge",         // 무료 Edge TTS
      "edge": {
        "enabled": true,
        "voice": "ko-KR-SunHiNeural",  // 한국어 여성 목소리
        "lang": "ko-KR",
        "rate": "+0%",            // 말하기 속도 (조절 가능)
        "pitch": "+0%"            // 음높이 (조절 가능)
      }
    }
  }
}

# 3. Gateway 재시작
openclaw gateway restart
```

**한국어 음성 옵션**:
- `ko-KR-SunHiNeural` - 여성 (밝고 친근함) ⭐
- `ko-KR-InJoonNeural` - 남성 (차분함)
- `ko-KR-BongJinNeural` - 남성 (뉴스 앵커 톤)
- `ko-KR-GookMinNeural` - 남성 (젊은 톤)

## 🎨 고급 설정

### 음성 속도 조절

```json
{
  "messages": {
    "tts": {
      "edge": {
        "rate": "+20%"   // 20% 빠르게
      }
    }
  }
}
```

**권장 값**:
- `-20%` ~ `+20%` (일상 대화)
- `-30%` (천천히, 이해하기 쉽게)
- `+30%` (빠르게, 시간 절약)

---

### 음높이 조절

```json
{
  "messages": {
    "tts": {
      "edge": {
        "pitch": "-10%"  // 낮은 목소리
      }
    }
  }
}
```

---

### 음성 메시지 받았을 때만 음성 응답

```json
{
  "messages": {
    "tts": {
      "auto": "inbound"  // 음성 메시지 받았을 때만 음성 응답
    }
  }
}
```

**사용 시나리오**:
- 텍스트 메시지 → 텍스트 응답
- 음성 메시지 → 음성 응답

---

### 긴 응답 자동 요약

```json
{
  "messages": {
    "tts": {
      "auto": "always",
      "summaryModel": "google/gemini-2.0-flash",  // 무료!
      "maxLength": 1500  // 1,500자 넘으면 요약
    }
  }
}
```

**효과**:
- 긴 답변(2,000자) → 요약(500자) → 음성 1분
- 짧은 답변(100자) → 그대로 → 음성 10초

---

### Slash 명령어로 실시간 제어

```
/tts on          # 음성 응답 켜기
/tts off         # 음성 응답 끄기
/tts inbound     # 음성 메시지 받았을 때만
/tts status      # 현재 설정 확인
/tts audio 안녕하세요   # 일회성 음성 생성
```
